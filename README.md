<h2>Three.js</h2>

<h3>소개</h3>
이 프로젝트는 Three.js Journey(https://threejs-journey.com/)를 참고하여 개발되었습니다. 3D 그래픽스를 효과적으로 구현하고, 사용자 경험을 최적화하기 위해 다양한 Three.js 기능을 활용했습니다.

<h3>주요 기능</h3>
- 3D 모델 렌더링 및 조작 기능
- Three.js Journey를 통한 심화 학습 및 기능 활용
- 사용자 상호작용 및 애니메이션 기능 최적화


<h3>설치 및 실행 방법</h3>
이 저장소를 클론합니다.

```git clone <repository-url>```
프로젝트 디렉토리로 이동한 후 필요한 종속성을 설치합니다.

```
cd <project-directory>
```

```
npm install
```

애플리케이션을 실행합니다.

```
npm start
```

<h3>기술 스택</h3>
Three.js - 3D 그래픽스 라이브러리
JavaScript 및 WebGL
